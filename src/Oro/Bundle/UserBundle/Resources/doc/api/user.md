## ACTIONS  

### get

{@inheritdoc}

### create

Create a new user record.

The created record is returned in the response.

{@inheritdoc}

{@request:json_api}
Example:

```JSON
{
   "data": {
      "type": "users",
      "attributes": {
         "username": "testapiuser",
         "email": "testuser@oroinc.com",
         "firstName": "Bob",
         "lastName": "Fedeson",
         "password": "Password000!"
      },
      "relationships": {
         "owner": {
            "data": {
               "type": "businessunits",
               "id": "1"
            }
         },
         "organization": {
            "data": {
               "type": "organizations",
               "id": "1"
            }
         }
      }
   }
}
```
{@/request}
