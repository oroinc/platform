# Oro\Bundle\IntegrationBundle\Api\Model\WebhookFormat

## ACTIONS

### get

Retrieve a specific webhook format record.

### get_list

Retrieve a collection of available webhook format records.

This resource provides a list of all available webhook formats that can be used when configuring webhooks.

## FIELDS

### id

The unique identifier of a resource.

### label

The human-readable name of the webhook format.
