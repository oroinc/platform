# Oro\Bundle\CommentBundle\Entity\Comment

## ACTIONS  

### get

Retrieve a specific comment record.

{@inheritdoc}

### get_list

Retrieve a collection of comment records.

{@inheritdoc}

### create

Create a new comment record.

The created record is returned in the response.

{@inheritdoc}

{@request:json_api}
Example:

```JSON
{
   "data": {
      "type": "comments",
      "attributes": {
         "message": "<p>test contact</p>"
      },
      "relationships": {
         "owner": {
            "data": {
               "type": "users",
               "id": "1"
            }
         },
         "organization": {
            "data": {
               "type": "organizations",
               "id": "1"
            }
         },
         "target": {
            "data": {
               "type": "emails",
               "id": "123"
            }
         }
      }
   }
}
```
{@/request}

### update

Edit a specific comment record.

The updated record is returned in the response.

{@inheritdoc}

{@request:json_api}
Example:

```JSON
{
   "data": {
      "type": "comments",
      "id": "11",
      "attributes": {
         "message": "<p>test contact</p>"
      },
      "relationships": {
         "owner": {
            "data": {
               "type": "users",
               "id": "1"
            }
         },
         "organization": {
            "data": {
               "type": "organizations",
               "id": "1"
            }
         },
         "target": {
            "data": {
               "type": "emails",
               "id": "123"
            }
         }
      }
   }
}
```
{@/request}

### delete

Delete a specific comment record.

{@inheritdoc}

### delete_list

Delete a collection of comment records.

{@inheritdoc}

### attachment
